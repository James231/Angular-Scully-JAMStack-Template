Angular Scully Example Template

This is an Angular project with Scully setup for JAMStack use. The repo structure is setup with example pages, layouts and custom components. Also has example Network and Cookie services ready to go (using dependency injection).

Example Pages:

  • Colors Example
  • Components Example
  • Event Bindings Example
  • Layout Example
  • Cookies Service Example
  • Network Service Example
